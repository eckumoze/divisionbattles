Division Battles — первый сплит 2026

1. Общие положения

Division Battles (DB) — любительская киберспортивная лига по Dota 2. Лига основана на системе дивизионов с повышением и понижением по итогам каждого сплита.

2. Формат турнира

2.1 Система дивизионов

  • Команды распределяются по дивизионам в соответствии с результатами.
  • По итогам сплита лучшие команды повышаются, худшие — понижаются.
  • Количество дивизионов зависит от числа участников.

2.2 Расписание

  • Одна встреча в неделю (Best of 3).
  • Капитаны команд самостоятельно согласовывают дату и время матча.

2.3 Плей-офф

  • Лучшие команды каждого дивизиона проходят в плей-офф.
  • Формат плей-офф определяется перед началом сплита.

3. Составы

  • В заявке команды может находиться до 10 игроков.
  • Минимальный состав для матча — 5 игроков.
  • Замена игроков возможна между матчами.
  • Один игрок может быть зарегистрирован только за одну команду в сплите.

4. Проведение матчей

  • Матчи проводятся на европейских серверах (Luxembourg / Stockholm / Vienna).
  • Режим — Captains Mode.
  • Победитель определяется по результатам Best of 3.
  • При технических проблемах — переигровка по решению администрации.

5. Призовой фонд

  • Призовой фонд первого сплита — зависит от числа команд.
  • Распределение призового фонда объявляется перед началом плей-офф.
  • Фонд может увеличиваться за счёт партнёрских взносов.

6. Дисциплина

  • Запрещены оскорбления, токсичное поведение и читерство.
  • Использование стороннего ПО для получения преимущества запрещено.
  • Администрация оставляет за собой право дисквалификации команды.
